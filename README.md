# redoor-devtool
redoor debugger

Debugger tool for Redoor library. See https://github.com/rubender/redoor





## Install

```yarn add redoor-devtool```

## Usage

### Run dev server

```npx redoor-devtool```

when open browser at http://localhost:8333
or
```npx redoor-devtool -o```
open browser `google-chrome` automatically
